# ESP32-P4 USB扩展屏幕项目文档索引

## 项目概述

<span style="color:red;">ESP32-P4 USB扩展屏幕</span>是一个让 ESP32-P4 开发板通过 USB 接口作为 Windows 操作系统外接显示器的项目。

| 项目 | 值 |
|:--|:--|
| 芯片 | ESP32-P4FN4 |
| IDF版本 | 5.4.0 |
| 屏幕分辨率 | 1024×600 |
| 刷新率 | 60 FPS |
| 触控点数 | 最多5点 |
| 音频采样率 | 48000 Hz |
| USB模式 | High-Speed OTG |

## 快速链接

### 构建命令

```bash
# 设置IDF环境
. ./export.sh

# 设置目标芯片
idf.py set-target esp32p4

# 烧录并监控
idf.py -p PORT flash monitor
```

### 关键配置

| 配置项 | 文件 | 默认值 |
|:--|:--|:--|
| VID/PID | `usb_descriptor.c` | 0x303A/0x2986 |
| 分辨率 | `usb_descriptor.c` | 1024×600 |
| JPEG质量 | `usb_descriptor.c` | Ejpg4 |
| 采样率 | `sdkconfig` | 48000 Hz |
